## Instalar el paquete

```bash
npm install @otrodigital/astro-i18n-next
```

## Dependencias peer

Los siguientes paquetes deben estar instalados en tu proyecto:

| Paquete | Versión | Requerido | Propósito |
|---------|---------|-----------|-----------|
| `astro` | `>= 5.0.0` | Sí | Framework Astro con soporte de enrutamiento i18n |
| `i18next` | `>= 23.0.0` | Sí | Motor de traducción |
| `js-yaml` | `>= 4.0.0` | Sí | Parseo de frontmatter YAML para contenido multilingüe |
| `marked` | `>= 9.0.0` | Opcional | Renderizado de Markdown para el cuerpo del contenido |

Instala las dependencias requeridas:

```bash
npm install astro i18next js-yaml
```

Si planeas usar el cargador de contenido multilingüe (`createMultilingualLoader`), también instala `marked`:

```bash
npm install marked
```
